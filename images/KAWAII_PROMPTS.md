# 🎀 Kawaii Cat Image Prompts — Cat Monopoly

Use these with any AI image generator (e.g. ChatGPT/DALL·E, Midjourney, Leonardo,
Bing Image Creator, Stable Diffusion). Generate each cat, save it as a PNG with the
**exact filename** shown, and drop it in this `images/` folder. The game picks it up
automatically (and falls back to the built-in SVG until you do).

---

## ✨ Shared style block (paste this in FRONT of every cat prompt)

> Kawaii chibi cat sticker, cute big sparkly eyes, tiny pink blush cheeks, soft
> rounded fluffy body, friendly happy expression, pastel storybook style, smooth
> cel shading, thick soft outline, centered front-facing portrait, plain solid
> background, high quality, consistent mascot style —

Then add the breed line below. For a clean game look, ask for **"transparent
background, square 512x512"** (or remove the background afterward).

### 💡 Tips for a matching set
- Generate all 18 in **one session** with the same style block so they share a look.
- Same framing every time (head + upper body, centered, facing forward).
- 512×512 PNG, transparent background = cleanest in-game.
- If a generator drifts in style, regenerate just that one cat.
