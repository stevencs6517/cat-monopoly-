CAT MONOPOLY — cat images folder
================================

Drop a PNG for each breed in THIS folder, using these EXACT filenames.
(Square images work best, e.g. 512x512. Transparent background looks cutest.)
If a file is missing, the game automatically shows the built-in kawaii SVG cat
instead, so nothing ever looks broken.

Required filenames (18):

  stray-kitten.png
  shorthair-cat.png
  black-cat.png
  tuxedo-cat.png
  ragdoll-cat.png
  garfield-cat.png
  ginger-cat.png
  sphinx-cat.png
  persian-cat.png
  bengal-cat.png
  scottish-fold.png
  siamese-cat.png
  devon-rex-cat.png
  maine-coon-cat.png
  angora-cat.png
  siberian-cat.png
  russian-blue.png
  tabby-cat.png

Tip: keep a consistent art style across all 18 (same framing, same cute
chibi proportions) so they look like a matching set in the game.
